> A mini-program for baidu contest
>
> Playing board game on phone at anywhere!

## Toolchain

- Taro@v2
- apollo-client
- React (hooks)
- TypeScript
- Lesscss

> Why use Taro@v2 instead of Taro@v3
>
> See [this issue](https://github.com/NervJS/taro/issues/7293).

## Install

```sh
yarn install
```

## Usage

```sh
yarn dev:swan
```
